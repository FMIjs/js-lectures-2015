# скриптове

Пишем програмата си във файл. След това подаваме името на файла като аргумент на `node` командата и по този начин го изпълняваме.

Изненадани сте, нали?

```sh
    $ cat hi.js
    function sayHi(name) {
      console.log('Hello, ' + name);
      }

      sayHi('Pencho');
    $ node hi.js
    Hello, Pencho
```
